# GPA Calculator Web App
A web application that allows students to:
- Add academic terms
- Add courses manually
- Track grade impact on GPA
- Forecast best-case GPA scenarios
- Custom the expected grades to forecast GPA in many cases.

## Tech Stack
- HTML
- CSS
- JavaScript

## Features
- Manual course input
- Dynamic GPA recalculation
- Term-based structure

## Future Improvements
- PDF syllabus parsing
